# Покер у Улукбека — исправленная версия 2.0

## Локальный запуск
```bash
corepack enable
corepack prepare pnpm@10.28.1 --activate
pnpm install
pnpm start
```
Откройте `http://localhost:3000`.

## Render
Build Command:
```bash
corepack enable && corepack prepare pnpm@10.28.1 --activate && pnpm install --frozen-lockfile=false
```
Start Command:
```bash
node server.js
```
После изменения обязательно: **Manual Deploy → Clear build cache & deploy**.

## Исправлено
- Удалён сломанный JavaScript внутри `<script src>`.
- Полностью пересобраны клиентские обработчики.
- Исправлена логика онлайн-комнаты и последовательность ходов.
- Исправлены ходы в игре на двоих, блайнды, колл/чек и переход на флоп.
- Действия доступны только игроку, чей сейчас ход.
- Старт доступен только создателю комнаты.
- Улучшена мобильная версия: компактный стол, крупные карты и закреплённая панель действий.
- Добавлены статусы подключения и переподключения.

Баланс виртуальный. Реальные платежи не подключены.

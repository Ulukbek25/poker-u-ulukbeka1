# Покер у Улукбека — онлайн

## Запуск локально
1. Установите Node.js 20.
2. В папке проекта выполните:
   npm install
   npm start
3. Откройте http://localhost:3000

## Размещение на Render
- Build Command: npm install
- Start Command: npm start
- Environment: Node
- Node version: 20

## Что уже работает
- Создание онлайн-комнат.
- Вход друзей по коду комнаты.
- Только реальные подключившиеся игроки, без ботов.
- Дилер и автоматическая демонстрационная раздача.
- Чат.
- Готовность игроков.
- Приватная отправка карт каждому игроку.

## Важно
Это технический прототип. Полные правила покера, ставки, победитель, база данных, регистрация, KYC, защита от мошенничества, реальные деньги и MBANK требуют дальнейшей серверной разработки и юридического согласования.

## Финальное размещение на Render
1. Загрузите содержимое архива в GitHub-репозиторий.
2. В Render выберите New → Blueprint.
3. Подключите репозиторий. Render прочитает `render.yaml` автоматически.
4. После публикации откройте выданный адрес сайта.

Проверено: `npm install`, запуск сервера, `/health` и загрузка главной страницы.
